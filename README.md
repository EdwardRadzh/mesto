# Проект 4: Место

## Описание
Это четвёртый обучающий проект на курсе фронтэнд разработчиков в Яндекс.Практикум

Работа над этим проектом позволяет изучить основы вёрстки.

В проекте использованы технологии/принципы:
* Методология БЭМ
* Flex-box
* grid
* Медиа-запросы
* JavaScript
* Git
